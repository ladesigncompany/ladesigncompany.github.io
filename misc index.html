Kamen Rider Kuuga Logo Translation

Created using Adobe Illustrator.
As a design experiment and challenge I tried translating the Japanese logo of a Japanese TV show into English while maintaining the same style. I found a font that resembled the original Japanese logo. I looked for features in the Japanese logo that can be applied to the English logo. For examples, I took the descenders you see at the bottom of each character in「クウガ」and applied them to the “K” and “A”. I then recreated the background icon using Illustrator’s 3D tools to get the bevel effect. I used gradients for the metallic effect on the bevel.
